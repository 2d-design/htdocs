Wallpaper Changer


Description:

This program fetches image links and feeds them to KLWP.
The wallpaper updates only when the previous one is seen.
It uses 500px when on wifi or ethernet connection and
local directories when there is slow or no connectivity.

Settings:

Interval: The time between two refreshes in minutes.
Search Term: Free text image description. Can be empty.
Photo Categories: Coma separated list of categories.
Consumer Key: Valid user authentication key.
Local Directories: Coma separated list of image directories.

How to use:

First install this program and KLWP.
Obtain a consumer key and fill in the settings.
In KLWP set $br(tasker, image)$ as wallpaper code.
Change the filters the way You like.
Sit back and enjoy the beautiful images!
